# 샘플 이미지 폴더

## 목적
디자인 작업 중 사용할 샘플 이미지를 저장하는 폴더입니다.

## 폴더 구조
```
/public/sample-images/
├── news/          # 뉴스 썸네일 이미지
├── profiles/      # 프로필 이미지
├── posts/         # 게시물 이미지
├── galleries/     # 갤러리 이미지
└── banners/       # 배너/이벤트 이미지
```

## 사용 방법
1. 각 카테고리별 폴더에 이미지 저장
2. 코드에서 `/sample-images/카테고리/파일명.jpg` 형식으로 참조
3. 예: `<img src="/sample-images/news/sample1.jpg" />`

## 주의사항
- 이 폴더는 디자인 작업용입니다
- 실제 배포 시에는 삭제하거나 실제 이미지로 교체하세요
- 이미지 파일명은 영문과 숫자만 사용하세요

## 권장 이미지 크기
- 뉴스 썸네일: 400x300px
- 프로필 이미지: 200x200px
- 게시물 이미지: 800x600px
- 갤러리 이미지: 1200x800px
- 배너 이미지: 1920x400px
