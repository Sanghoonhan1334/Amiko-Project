# .env.local 파일 설정 가이드

## 목적
디자인 작업 중 Supabase 연결 없이 앱을 실행하기 위한 더미 환경 변수 설정

## 파일 위치
`/amiko_new/.env.local`

## 내용
```env
# Supabase 설정 (디자인 작업용 더미 값)
NEXT_PUBLIC_SUPABASE_URL=https://dummy.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=dummy_key_for_design_work_only

# 기타 환경 변수
NEXT_PUBLIC_USE_MOCK_DATA=true
```

## 주의사항
- 이 파일은 `.gitignore`에 포함되어 Git에 커밋되지 않습니다
- 실제 배포 시에는 진짜 Supabase 값으로 교체해야 합니다
- 디자인 작업 완료 후 삭제하거나 실제 값으로 업데이트하세요

## 적용 방법
1. `.env.local` 파일 생성
2. 위 내용 복사
3. 개발 서버 재시작 (`npm run dev`)
