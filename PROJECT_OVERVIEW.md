# Amiko 프로젝트 개요

## 📋 프로젝트 소개

**Amiko**는 한국과 라틴아메리카 간의 문화 및 교육 교류를 위한 플랫폼입니다.

- **기술 스택**: Next.js (App Router), Supabase, Tailwind CSS
- **결제 시스템**: Toss Payments (구현 완료), PayPal (구조 준비 완료)
- **주요 목적**: 문화 교류, 언어 학습, 커뮤니티 형성

## 🏗️ 현재 구현된 주요 모듈

### 1. 인증 시스템
- Supabase Auth 기반 로그인/회원가입
- 전화번호 인증 (SMS/WhatsApp)
- 생체인증 (WebAuthn)
- 이메일 인증

### 2. 커뮤니티
- 게시글 작성/수정/삭제
- 댓글 시스템
- 좋아요/반응
- 갤러리 시스템
- 자유게시판
- 인기 게시물

### 3. 팬존
- 아이돌 사진 갤러리
- 팬아트 공유
- K-Pop 뉴스
- 스토리 기능

### 4. 퀴즈 시스템
- 한국어 레벨 테스트
- 아이돌 포지션 퀴즈
- MBTI 테스트

### 5. 포인트 시스템
- 일일 활동 포인트
- 랭킹 시스템
- 이벤트 포인트
- 추천인 포인트

### 6. 이벤트
- Zep 이벤트 관리
- 추첨 시스템
- 월간 포인트 이벤트

### 7. 예약 시스템
- 상담사/파트너 예약
- 일정 관리
- 예약 승인/거절

### 8. 결제 시스템
- **Toss Payments**: 한국 사용자용 (구현 완료)
- **PayPal**: 라틴아메리카 사용자용 (구조 준비 완료, 구현 예정)

### 9. 번역 시스템
- OpenAI GPT 번역
- Google Translate API
- Mock 번역 (개발용)

### 10. 알림 시스템
- 웹 푸시 알림 (VAPID)
- 이메일 알림
- SMS/WhatsApp 알림 (Twilio)

### 11. 관리자 시스템
- 사용자 관리
- 결제/예약 통계
- 콘텐츠 관리
- 이벤트 관리

## 🚀 앞으로 구현할 핵심 기능

### 1. PayPal 결제 시스템 (마리아 담당)
- **상태**: 기본 API 구조와 타입 정의 완료
- **위치**: `src/app/api/paypal/`
- **작업 내용**:
  - PayPal API 완전 연동
  - 라틴아메리카 결제 플로우 구현
  - KYC 처리
  - 자동 영수증 이메일
  - 웹훅 처리 강화

### 2. 법적 정책 강화 (법률 전문가 담당)
- **작업 내용**:
  - 미성년자 보호 정책
  - 개인정보 처리방침 개선
  - 약관/면책 조항 작성
  - 회원가입/로그인/결제 전 동의 문구
  - GDPR 준수 사항

### 3. 강의/수업 탭 (마리아 담당)
- 교육 콘텐츠 관리
- 수강 시스템
- 강의 일정 관리
- 수료증 발급

### 4. 라틴아메리카 사용자용 UI
- 스페인어/포르투갈어 UI
- 지역별 결제 옵션
- 문화적 적응

## 📁 주요 디렉토리 구조

```
src/
├── app/
│   ├── api/              # API 라우트
│   ├── main/             # 메인 앱
│   ├── community/        # 커뮤니티
│   ├── admin/            # 관리자
│   └── ...
├── components/
│   ├── main/app/         # 메인 앱 컴포넌트
│   ├── auth/             # 인증 컴포넌트
│   └── ...
├── lib/                  # 유틸리티
└── types/                # 타입 정의
```
